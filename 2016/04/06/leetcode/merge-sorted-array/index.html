leetcode:88--Merge Sorted Array

发表于 2016-04-06   |   分类于 Python ， leetcode ， list   |  

题目描述：合并排序数组
给定两个排过序的数组nums1和nums2，将两个数组合并为一个整体排序的数组。
nums1 = [1,3,5,0,0,0,0]
nums2 = [2,4,6,8]
合并后为：[1,2,3,4,5,6,7,8]

solution：

算法思想：
由于nums1和nums2都是排过序的，所以从后向前比较，将较大的一个放在最后的位置。

空间复杂度o(1),时间复杂度o(m+n)

Python Code: 


class Solution(object):
    def merge(self, nums1, m, nums2, n):
        #last位于list最后一个位置, i是nums1的最后一个位置,j是nums2的最后一个位置
        last = m + n - 1
        i = m - 1
        j = n - 1
        while i >= 0 and j >= 0:
            if nums1[i] > nums2[j]:
                #大的数放在最后的位置
                nums1[last] = nums1[i]
                #此时i向前移动一位,与j所在位置的数字比较
                i -= 1
                last -= 1
            else:
                nums1[last] = nums2[j]
                j -= 1
                last -= 1

        #此处还要加上对while j >= 0的判断,而不需要加上对while i >= 0的判断,因为,i本         
        #身就在nums1里面,j在nums2里面,要对nums2的数判断完加入nums1
        while j >= 0:
            nums1[last] = nums2[j]
            j -= 1
            last -= 1

if __name__ == "__main__":
    nums1 = [1,3,5,7,9,0,0,0,0]
    nums2 = [2,4,6,8]
    Solution().merge(nums1, 5, nums2, 4)
    print nums1

My Problem:

开始的时候比较疑惑,为什么要加一个while j >= 0的判断?

My Answer

不需要加上对while i >= 0的判断,因为,i本身就在nums1里面,j在nums2里面,要对nums2的数判断完加入nums1
