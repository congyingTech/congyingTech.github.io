  1. 显示遇到了Error when reading from SSL socket connection：SSLsocket问题

    This site requires JavaScript to be enabled
    Also I upgraded the privileges on the “vmware” account on the ILO/DRAC. If there is an existing Virtual Media session running that needs to be stop, you can manually kill it.

Here are the steps:
1.) SSH into (drivername)-ilo
2.) credentials: vmware/######
3.) run the command “racadm racreset”

Running racadm racreset will kill all sessions connecting to esx170-ilo. Give it 2 minutes to reboot and you can log back into the ILO via HTTP/HTTPS/SSH.. etc.
