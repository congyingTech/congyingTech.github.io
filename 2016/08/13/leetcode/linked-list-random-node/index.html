leetcode:382. Linked List Random Node

发表于 2016-08-13   |   分类于 Python ， leetcode ， 蓄水池算法   |     |   阅读次数

题目描述:
给定一个单链表，返回链表中一个随机节点的值。每一个节点应该被等可能的抽取。
进一步思考：
如果链表很大并且长度未知呢？你可以不使用额外的空间高效地解决此问题吗？

Solution

算法思想:
这是一个蓄水池问题,看了看解析,了解到蓄水池抽样的算法:
1.长度为1时,抽到这个元素的概率为100%,所以一定返回这个元素;
2.抽取前n个元素时,抽取的概率相等,为1/n;
3.抽取第n+1个元素时,若random.randint(0, n)等于0，则返回值替换为第n+1个元素，其概率为1/(n+1)；
否则，抽取的依然是前n个元素，其概率为1/n * n/(n+1) = 1/(n+1)
